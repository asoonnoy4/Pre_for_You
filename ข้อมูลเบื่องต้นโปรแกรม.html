Tool Bar

เป็นหน้างต่างที่ใช้ในเก็บเครื่องมือในการตัดต่อประกอบด้วย(เรียงจากด้านบน)
1. Selection Tool : ใช้เลือกคลิปที่ต้องการทำงาน
2. Track Select Tool : ใช้เลือกคลิปทั้งหมดที่อยู่ บน Track เดียวกัน
3. Ripple Edit Tool : ใช้ย่อ/ขยายความยาวของ คลิปโดยมีผลต่อความยาวทั้งหมด
4. Razor Tool : ใช้ตัดแบ่งคลิปออกเป็นคลิปย่อย
5. Slip Tool : เลื่อนฉากในคลิปโดยไม่กระทบต่อ คลิปรอบข้างและความยาวทั้งหมด
6. Pen Tool : ใช้จัดการกับคีย์เฟรมบนคลิป
7. Hand Tool : ใช้เลื่อนมุมมองบนหน้าต่าง Time line
8. Type Tool : ใช้ในการใส่ข้อความลงในวีดีโอ
